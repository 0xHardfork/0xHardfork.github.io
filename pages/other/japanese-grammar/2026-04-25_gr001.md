## 1. ～は～です

| 項目 | 内容 |
|------|------|
| 文法 | 【名】は【名】です |
| 接続 | 名詞＋は＋名詞＋です |
| 意味 | ～是～ |
| Level | N4+N3 |

**助记**: 「は」是主题标记助词，「です」是"是"的礼貌体。想象自己在自我介绍："我**は**学生**です**"——把主题摆出来，然后下定义。

**例句**:
1. 私は学生です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 我是学生。
2. 田中さんは日本人です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 田中先生是日本人。
3. これは辞書です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 这是字典。

**使用ポイント:**
- 「は」读作「wa」而不是「ha」，这是初学者最容易犯的错误
- 「です」是礼貌体，日常会话中必不可少
- 可以省略主语，但「は」标记的主题通常不省略

## 2. ～は～ではありません

| 項目 | 内容 |
|------|------|
| 文法 | 【名】は【名】ではありません |
| 接続 | 名詞＋は＋名詞＋ではありません |
| 意味 | ～不是～ |
| Level | N4+N3 |

**助记**: 「では」= 把范围框出来，「ありません」= 不存在。两个加起来就是"在那个范围里不存在"→"不是"。

**例句**:
1. 私は先生ではありません。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 我不是老师。
2. これは本ではありません。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 这不是书。
3. 山田さんは医者ではありません。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 山田先生不是医生。

**使用ポイント:**
- 这是「～は～です」的否定形式
- 「ではありません」是礼貌的否定，口语中常缩略为「じゃありません」
- 语气比「じゃない」更正式

---

## 3. ～は～じゃありません

| 項目 | 内容 |
|------|------|
| 文法 | 【名】は【名】じゃありません |
| 接続 | 名詞＋は＋名詞＋じゃありません |
| 意味 | ～不是～（口语） |
| Level | N4+N3 |

**助记**: 「じゃ」是「では」的口语缩略版。想象"では"说太快就变成了"じゃ"——嘴巴偷懒的结果。

**例句**:
1. 彼は学生じゃありません。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 他不是学生。
2. 今日は月曜日じゃありません。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 今天不是星期一。
3. あれは私の傘じゃありません。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 那不是我的伞。

**使用ポイント:**
- 「じゃありません」和「ではありません」意思完全相同，只是语体不同
- 日常会话更常用「じゃありません」
- 更随意的说法是「じゃない」（普通体）

## 4. ～は～ですか

| 項目 | 内容 |
|------|------|
| 文法 | 【名】は【名】ですか |
| 接続 | 名詞＋は＋名詞＋ですか |
| 意味 | ～是～吗？ |
| Level | N4+N3 |

**助记**: 日语的疑问句超简单——在句尾加「か」就行！就像中文在句尾加"吗"。「か」= 疑问开关。

**例句**:
1. あなたは田中さんですか。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 你是田中先生吗？
2. それは新聞ですか。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 那是报纸吗？
3. 佐藤さんは会社員ですか。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 佐藤先生是公司职员吗？

**使用ポイント:**
- 肯定回答：はい、～です
- 否定回答：いいえ、～ではありません
- 日语疑问句不需要改变语序，只需在句尾加「か」

## 5. ～の～（从属关系）

| 項目 | 内容 |
|------|------|
| 文法 | 【名】の【名】 |
| 接続 | 名詞＋の＋名詞 |
| 意味 | ～的～（表示从属的机构、国家或属性） |
| Level | N4+N3 |

**助记**: 「の」就是中文的"的"。A**の**B = A**的**B。日语里最常用的助词之一，出现频率极高。

**例句**:
1. 私は東京大学の学生です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 我是东京大学的学生。
2. これは日本の車です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 这是日本的车。
3. 田中さんは三年生の学生です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 田中是三年级的学生。

**使用ポイント:**
- 「の」连接两个名词，前面的名词修饰后面的名词
- 可以表示所属、产地、身份等多种关系
- 可以连续使用：東京大学の日本語学科の学生

## 6. ～さん

| 項目 | 内容 |
|------|------|
| 文法 | ～さん |
| 接続 | 名前・名字＋さん |
| 意味 | 在人名后加上さん，表示敬意（先生/女士） |
| Level | N4+N3 |

**助记**: 「さん」是日语中最百搭的敬称，男女老少通用。就像中文的"先生/女士/同学"合体版。注意：绝对不能用在自己名字后面！

**例句**:
1. 田中さんは先生です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 田中先生是老师。
2. 山田さん、おはようございます。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 山田先生/女士，早上好。
3. 佐藤さんはどこですか。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 佐藤先生/女士在哪里？

**使用ポイント:**
- 绝不能在自己名字后面加「さん」
- 比「様」（さま）更日常、亲切
- 也可以用在店名后面：花屋さん（花店老板）

## 7. ～も～です

| 項目 | 内容 |
|------|------|
| 文法 | 【名】も【名】です |
| 接続 | 名詞＋も＋名詞＋です |
| 意味 | ～也是～ |
| Level | N4+N3 |

**助记**: 「も」= "也"。它替代了「は」，自带"也"的含义。想象：A**は**学生です → B**も**学生です（B也是学生）。

**例句**:
1. 私も学生です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 我也是学生。
2. 田中さんも日本人です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 田中先生也是日本人。
3. これも辞書です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 这个也是字典。

**使用ポイント:**
- 「も」替代「は」使用，不能写成「はも」
- 用于前面已有同类事物被提及的情况
- 否定：～も～ではありません（～也不是～）

## 8. これ/それ/あれ は～です

| 項目 | 内容 |
|------|------|
| 文法 | これ/それ/あれ は【名】です |
| 接続 | 指示代词＋は＋名詞＋です |
| 意味 | 这个/那个/那个（远处）是～ |
| Level | N4+N3 |

**助记**: こ=近（靠近我）、そ=中（靠近你）、あ=远（离我们都远）。三级距离系统，像三个同心圆：「こ」在圆心，「そ」在中间，「あ」在最外。

**例句**:
1. これは本です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 这是书。
2. それはペンですか。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 那是笔吗？
3. あれは学校です。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 那个（远处的）是学校。

**使用ポイント:**
- こ系列（これ、この、ここ）：离说话人近的事物
- そ系列（それ、その、そこ）：离听话人近的事物
- あ系列（あれ、あの、あそこ）：离双方都远的事物

## 9. ～だれ/どなた ですか

| 項目 | 内容 |
|------|------|
| 文法 | ～はだれ（どなた）ですか |
| 接続 | 名詞＋は＋だれ/どなた＋ですか |
| 意味 | ～是谁？ |
| Level | N4+N3 |

**助记**: 「だれ」= 谁（普通），「どなた」= 哪位（礼貌）。想象对陌生人说"哪位？"比"谁？"更有礼貌——「どなた」就是这样的升级版。

**例句**:
1. あの人はだれですか。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 那个人是谁？
2. 先生はどなたですか。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 老师是哪位？
3. これはだれの本ですか。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 这是谁的书？

**使用ポイント:**
- 「だれ」用于朋友间的日常对话
- 「どなた」用于需要表示敬意的场合
- 可以和「の」搭配：だれの～（谁的～）

## 10. ～なんですか

| 項目 | 内容 |
|------|------|
| 文法 | ～は なんですか |
| 接続 | 名詞＋は＋なん＋ですか |
| 意味 | ～是什么？ |
| Level | N4+N3 |

**助记**: 「なん」= 什么（なに的口语缩略）。当后面跟「です」时，「なに」变成「なん」——因为发音更顺口。

**例句**:
1. これはなんですか。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 这是什么？
2. お仕事はなんですか。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 您的工作是什么？
3. 趣味はなんですか。 [🔊](./2026-04-25_gr001-examples.mp3)
   > 爱好是什么？

**使用ポイント:**
- 「なに」在「です」「で」「の」前面变成「なん」
- 「なに」在其他助词前保持原形：なにを、なにが
- 回答时直接用名词：「本です」（是书）
