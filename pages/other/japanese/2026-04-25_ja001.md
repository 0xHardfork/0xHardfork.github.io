## 1. 高校

| 項目 | 内容 |
|------|------|
| 単語 | 高校 |
| 重音 | ⓪ |
| 假名 | こうこう [🔊](./2026-04-25_ja001-words.mp3) |
| 词性 | 名 |
| 意思 | 高中 |
| Level | 1-N4+N5 |

**例句**: 私（わたし）は高校（こうこう）の時（とき）、サッカー部（ぶ）に入（はい）っていました。 [🔊](./2026-04-25_ja001-sentences.mp3)
> 我高中的时候加入了足球社。

**助记**: 高＝高等的，校＝学校 → 高等学校 = 高中。注意：日语的「高校」指高中，和中文的「高校」（大学）不同。

## 4. 昭和

|------|------|

| 重音 | ⓪① |
| 假名 | しょうわ [🔊](./2026-04-25_ja001-words.mp3) |
| 词性 | 名 |
| 意思 | （日本年号）昭和 |
| Level | 4-N1::2-中频 |

## 📖 今日短文

[🔊 短文朗读](./2026-04-25_ja001-story.mp3)

### 日本語

ある日（ひ）、高校（こうこう）の理系（りけい）のクラスで、先生（せんせい）が言いました。「今（いま）から班（はん）を作（つく）って、外（そと）で活動（かつどう）します。田中（たなか）君（くん）と佐藤（さとう）さん、同（おな）じ班（はん）になってください。」

### 中文翻译

有一天，在高中的理科班级里，老师说：“现在我们要分组在外面进行活动。小田中和佐藤女士，请你们分成一组。”

两人走了一公里左右来到海边，捡了几个漂亮的贝壳。在这期间，他们谈论了昭和和明治时代的历史。“以前的政治家经常结党呢，”小田中说。“但是，好像含糊不清的问题很多，陷入恶性循环的情况也频发，”佐藤女士回答道。

之后，因为肚子饿了，两人去了食堂。小田中点了一份大碗的拉面。佐藤女士吃了三百克肉，和一个甜味很浓的甜点。“这拉面量太多了，很难吃完啊，”小田中笑着说。

在食堂换气扇的转动声中，两人开心地笑了很多次。然后，他们在笔记本上画了一个圆，感受到了一小撮希望，互相诉说了成为著名科学家的梦想。回去的时候，他们前往了公寓三楼的图书室。这一个月来，课程都非常容易懂，小花子也很高兴。
