# SAST - 静态应用安全测试

## 概述

SAST (Static Application Security Testing) 通过分析源代码、字节码或二进制文件来发现安全漏洞。

## 子主题

- [CodeQL](./codeql.html) - GitHub的代码查询语言
- [大模型SAST](./llm-sast.html) - 基于大语言模型的安全分析

## 优势

✅ 早期发现漏洞  
✅ 覆盖所有代码路径  
✅ 可集成到CI/CD  
✅ 无需运行应用

## 局限性

❌ 可能产生误报  
❌ 无法检测运行时问题  
❌ 配置复杂度高
